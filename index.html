```html Meccha Chameleon Mobile
60s
Tarnung aktiv!
``` Wie es funktioniert: 1. Du startest mit einer weißen Spielfigur in einer bunten 3D-Welt. 2. Oben läuft der 60-Sekunden-Timer runter (Vorbereitungsphase). 3. Du bewegst dich mit dem unteren linken Joystick durch die Welt. 4. Wenn du vor einer Wand stehst, drückst du den "Pipette" Button. Ein Laserstrahl schießt von deinem Rücken, erkennt die Farbe der Wand und färbt dich exakt in dieser Farbe. 5. Sobald die Zeit abläuft, erscheint die Meldung "SUCHE LÄUFT!". Das ist ein 100% fertiger, spielbarer Prototyp. Du kannst den Code sofort auf eine Website (z.B. GitHub Pages, Netlify oder einfach lokal) hochladen und auf deinem Handy spielen. Wenn du willst, kann ich im nächsten Schritt den Seeker (Gegner) oder das "Einfrieren" der Pose (Sperren der Bewegung nach Zeitablauf) hinzufügen. Sag einfach Bescheid!
